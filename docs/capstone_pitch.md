# Capstone Pitch

Global AI Conversation Analytics Platform is a Google Trends-style dashboard for understanding what people around the world ask AI.

It turns millions of raw AI conversations into country, language, topic, sentiment, and trend insights. The platform supports data ingestion from CSV, JSON, and Parquet, cleans and enriches the data using Pandas and NLP, stores the results in PostgreSQL, and presents the findings in a Streamlit dashboard with maps, charts, translation, AI topic extraction, Ask the Dataset, and ElevenLabs voice briefings.

This project demonstrates data engineering, NLP, dashboard development, API integration, geospatial analytics, privacy-aware AI design, and storytelling with data.
