# Capstone Demo Script

## Goal
Show that this is a Google Trends-style platform for global AI conversation insights.

## 5–7 Minute Flow

1. Introduce the problem: millions of AI conversations are hard to understand without analytics.
2. Explain the data pipeline: ingest CSV/JSON/Parquet, clean with Pandas, enrich with NLP, store in PostgreSQL.
3. Show Global Overview: conversations, countries, languages, top topics, trending topics.
4. Show Interactive World Map: click a country and explain how the dashboard updates.
5. Show Country Analysis: top topics, sentiment, common questions, language distribution.
6. Show Language Analysis: compare topic differences by language.
7. Show Sentiment Dashboard: positive, neutral, negative trends.
8. Show Wow Factor page: Global Curiosity Index, Trend Timeline, Heatmap, Topic Cloud.
9. Ask the Dataset: ask a natural-language analytics question.
10. Show Translation Lab: translate a safe summary to English and local language.
11. Show Voice Briefing Studio: generate or play an ElevenLabs country briefing.
12. Close with ethics: raw sensitive data is not displayed; insights are aggregated and safe.
